# 🤝 Como Contribuir

Obrigado pelo interesse em contribuir com o GhostCommit! 

## 🚀 Início Rápido

1. **Fork** o repositório
2. **Clone** seu fork: `git clone https://github.com/danielfalcaodf/GhostCommit.git`
3. **Instale** dependências: `npm install`
4. **Crie** uma branch: `git checkout -b feature/sua-feature`
5. **Desenvolva** e **teste** suas mudanças
6. **Commit**: `git commit -m "feat: sua nova feature"`
7. **Push**: `git push origin feature/sua-feature`
8. **Abra** um Pull Request
