## 🎯 Funcionalidades em Destaque

### ⭐ Anotação de Origem (Git Blame)
**Status**: 🚧 Em Desenvolvimento

Uma das funcionalidades mais importantes do GhostCommit, permitindo:
- 📍 Ver commit, autor e data para cada linha alterada
- 🎨 Visualização colorida por autor
- ⏰ Timeline das mudanças
- 🔍 Navegação pelo histórico de linha

**Documentação**: [Requisitos Funcionais](functional-requirements.md#2-📍-anotação-de-origem-git-blame)

### 🔍 Comparação Avançada de Commits
**Status**: ✅ Implementado

- Seleção intuitiva de commits, branches e tags
- Filtros dinâmicos e busca em tempo real
- Visualização estatística de mudanças
- Estado persistente entre navegações

**Documentação**: [Funcionalidades Git](git-features.md)

### 📊 Visualização de Diffs
**Status**: ✅ Implementado

- Monaco Editor com syntax highlighting
- Suporte a 30+ linguagens
- Interface responsiva e tema dark
- Navegação fluida entre arquivos

**Documentação**: [Funcionalidades Git](git-features.md#4-📝-visualizador-de-diff)
